# Codora CLI Companion

The Codora CLI Companion extension seamlessly integrates the Codora CLI (a fork of Gemini CLI) into your VS Code environment. This integration enables Codora CLI to access active information like your open files.

# Features

- Open Editor File Context: Codora CLI gains awareness of the files you have open in your editor, providing it with a richer understanding of your project's structure and content.

- Selection Context: Codora CLI can easily access your cursor's position and selected text within the editor, giving it valuable context directly from your current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by Codora CLI directly within the editor.

- Launch Codora CLI: Quickly start a new Codora CLI session from the Command Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Codora CLI: Run" command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- Codora CLI (installed separately) running within the VS Code integrated terminal
